# E‑commerce Frontend – TechStore

A modern, responsive single-page storefront called **TechStore**, showcasing premium electronics with a product grid, cart sidebar, search, smooth scrolling navigation, and basic UI interactions — all built with plain HTML, CSS, and JavaScript.

## Project Structure

- **html/index.html** – Main page layout, sections (hero, stats, features, products, testimonials, newsletter, content, footer) and DOM hooks for JS.
- **css/style.css** – Global styling, responsive layout, animations, and component styles (cards, grid, cart sidebar, search, mobile menu, etc.).
- **js/app.js** – Product data, cart logic, search functionality, navigation behavior, section animations, and mobile menu handling.

## Getting Started

1. **Clone / copy the project** into your local machine.
2. Open `html/index.html` directly in a browser (double-click or drag into Chrome/Edge/Firefox), or serve the folder with a simple static server.
3. Make sure you have an active internet connection so the Font Awesome CDN loads correctly.

## Customization

- **Products**: Edit or extend the `products` array in `js/app.js`.
- **Branding / text**: Update headings, copy, and links in `html/index.html`.
- **Colors & layout**: Adjust gradients, spacing, and component styles in `css/style.css`.

## Technologies Used

- HTML5
- Modern CSS (flexbox, grid, responsive design, animations)
- Vanilla JavaScript (DOM manipulation, event handling, basic state management)
